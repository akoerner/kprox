Change Store Key

Re-encrypts all credentials with a new key. Store must be unlocked. Min 8 chars for key-only mode; min 4 chars for PIN+TOTP mode.

Storage Location

The credential store database can be stored in built-in flash (NVS) or on the SD card. When the store is unlocked, switching location migrates the database automatically.

Danger Zone

Wipe deletes all credentials and the key check. Format erases the SD card database file. Both cannot be undone.

Credential Store Gate

Controls what is required to unlock the credential store. Off — symmetric key only (existing behaviour). Key + TOTP — enter the key first, then a live TOTP code. TOTP only — the TOTP code is the unlock; no separate key needed. If NTP is unavailable the store cannot be unlocked.

Required when leaving TOTP-only mode. All credentials will be re-encrypted with this key.
Required for Key+TOTP and TOTP-only modes. Store this separately — it is not recoverable from the device.
Scan with any TOTP app (Google Authenticator, Aegis, etc.)
URI
CS Encryption Key

The symmetric key that encrypts all stored credentials. Set this when using Off or Key+TOTP mode. If the store is currently unlocked, all credentials will be re-encrypted with the new key immediately. If locked, the new key takes effect on next unlock (the old key is discarded).

Scheduled Tasks

Tasks fire when the device clock matches the configured time. Leave the date-time at epoch (1970-01-01T00:00) to match any date — only the time components that matter need to be non-zero. Requires WiFi + NTP sync.

Add Task

Any valid KProx token string.
Year 0 (1970) = match any year, month 0 = any month, day 0 = any day.

Keymap Editor

Custom keymaps remap characters to different HID key + modifier combinations. The built-in en layout cannot be modified or deleted.

Char HID Key (dec) Modifier Mod flags Del
Modifier: 0=none 2=Shift 4=Alt 8=GUI 64=AltGr. Combine by adding.
Parses JSON and refreshes the table above.

Keymap JSON Format

Each entry maps a Unicode character to a HID usage ID and modifier byte. mod: 2=Shift, 4=Alt, 8=GUI, 64=AltGr. 

{"id":"my-layout","name":"My Custom Layout","map":[{"char":"y","key":29,"mod":0},{"char":"@","key":20,"mod":64}]}
