Register Management

Reference

Hardware Controls

  • Single Click: Play the currently active register
  • Double Click: Cycle to the next register and set it as active
  • Hold 2s: Toggle halt/resume all operations
  • Hold 5s: Delete all registers (emergency reset)
  • LED Feedback: The LED blinks to indicate the current active register number

Transport: BLE+USB both   BLE only   — no HID output

Key Token Arguments BLE+USB

  • {KEY} — tap (press + release immediately)
  • {KEY ms} — hold for ms milliseconds e.g. {ENTER 1000}
  • {KEY press} — press down only, no auto-release
  • {KEY release} — release only (use after press)
  • Example: {SHIFT press}{a}{b}{c}{SHIFT release} — types ABC

Basic Keys BLE+USB

  • {ENTER} / {RETURN}   {TAB}   {ESC} / {ESCAPE}   {SPACE}
  • {BACKSPACE} / {BKSP}   {DELETE} / {DEL}   {INSERT}
  • {LEFT} {RIGHT} {UP} {DOWN}   {HOME} {END} {PAGEUP} {PAGEDOWN}
  • {PRINTSCREEN} / {PRTSC} / {SYSRQ}   {APPLICATION} / {MENU} / {APP}

Modifier Keys BLE+USB

  • {GUI} / {MOD} / {WIN} / {CMD} / {SUPER} / {WINDOWS} — all aliases
  • {CTRL} / {LCTRL}   {RCTRL}   {ALT} / {LALT}   {RALT} / {ALTGR}
  • {SHIFT} / {LSHIFT}   {RSHIFT}

Lock / Toggle Keys BLE+USB

  • {CAPSLOCK} / {CAPS}   {NUMLOCK}   {SCROLLLOCK} / {SCRLK}   {PAUSE} / {PAUSEBREAK}

Function Keys

  • {F1}–{F12} BLE+USB
  • {F13}–{F24} BLE only (USB host support varies)

Numpad Keys BLE+USB

  • {KP0}–{KP9}   {KPENTER}   {KPPLUS}   {KPMINUS}   {KPMULTIPLY}/{KPSTAR}
  • {KPDIVIDE}/{KPSLASH}   {KPDOT}/{KPDECIMAL}

Media / Consumer Keys BLE+USB

HID Consumer Control report. No press/release/hold args. Produces correct kernel events (KEY_PLAYPAUSE etc.) visible in evtest.

  • {MUTE}   {VOLUMEUP}/{VOLUP}   {VOLUMEDOWN}/{VOLDOWN}
  • {PLAYPAUSE}   {NEXTTRACK}/{NEXT}   {PREVTRACK}/{PREV}   {STOPTRACK}/{STOP}
  • {CALCULATOR}/{CALC}   {EMAIL}   {MYCOMPUTER}   {WWWHOME}/{BROWSER}
  • {WWWSEARCH}   {WWWBACK}   {WWWSTOP}   {BOOKMARKS}   {MEDIASEL}

System Power / Sleep Keys BLE+USB

HID Generic Desktop / System Control report. No press/release/hold args.

  • {SYSTEMPOWER} / {SYSPOWER} / {POWERDOWN} — power down
  • {SYSTEMSLEEP} / {SYSSLEEP} — sleep
  • {SYSTEMWAKE} / {SYSWAKE} / {WAKEUP} — wake up

RELEASEALL BLE+USB

  • {RELEASEALL} — immediately sends zeroed keyboard + consumer + system reports, releasing all held keys

System / Device Control — no HID output

  • {HALT} — stop all execution   {RESUME} — resume
  • {BLUETOOTH_ENABLE} / {BLUETOOTH_DISABLE}   {USB_ENABLE} / {USB_DISABLE}
  • {WIFI ssid password} — connect to WiFi
  • {SINKPROX} — flush and execute the SinkProx buffer

Timing Tokens

  • {SLEEP 1000} - Wait 1000 milliseconds
  • {SLEEP {RAND 500 2000}} - Wait random time between 500-2000ms
  • {SCHEDULE 14:30} - Wait until 14:30 wall-clock time (uses NTP). Blocks until the device clock reaches that time.
  • {SCHEDULE 09:00:00} - Wait until 09:00:00 (with seconds precision)
  • SCHEDULE requires WiFi + NTP sync. The device syncs time on WiFi connect. Times are UTC unless your timezone offset is configured.
  • Example: {SCHEDULE 08:00}Good morning!{ENTER} - Types a message every day at 8 AM when the register loops
  • Example: {LOOP}{SCHEDULE 12:00}{CHORD GUI+L}{SLEEP 60000}{ENDLOOP} - Lock screen at noon daily

Loop Tokens

  • {LOOP}{ENDLOOP} - Infinite loop
  • {LOOP 5000}{ENDLOOP} - Loop for 5000 milliseconds (5 seconds)
  • {LOOP i 1 1 5}{ENDLOOP} - Loop variable i from 1 to 5, increment by 1
  • {LOOP j 10 -2 0}{ENDLOOP} - Loop variable j from 10 to 0, decrement by 2
  • {ENDLOOP} - End loop marker
  • {BREAK i 3} - Break out of loop when variable i equals 3

Input Control Tokens

  • {CHORD CTRL+C} - Press Ctrl+C
  • {CHORD ALT+TAB} - Press Alt+Tab
  • {CHORD SHIFT+A} - Press Shift+A
  • {CHORD GUI+L} - Press Windows/Cmd+L (GUI = Windows/Cmd key)
  • {CHORD CTRL+SHIFT+ESC} - Multiple modifiers (Task Manager)
  • {CHORD ALT+F4} - Alt+F4 (Close window)
  • {CHORD CTRL+ALT+DELETE} - Three-key combination
  • {CHORD F1} - Function keys
  • {CHORD SHIFT+F10} - Shift + Function key
  • {CHORD GUI+R} - Windows+R (Run dialog)
  • {CHORD CTRL+SHIFT+N} - Ctrl+Shift+N (New incognito window)
  • {CHORD MOD+L} — lock screen (MOD/GUI/WIN/CMD/SUPER all alias the GUI key)
  • {CHORD RALT+E} — AltGr+E (€ on European layouts)
  • {CHORD GUI+SPACE} — Spotlight (macOS) / launcher
  • Full modifier alias list: CTRL/LCTRL/RCTRL · ALT/LALT/RALT/ALTGR · SHIFT/LSHIFT/RSHIFT · GUI/MOD/WIN/CMD/SUPER/WINDOWS
  • {CHORD} works on BLE+USB

Raw HID Control Tokens

  • {HID 0x04} - Send keycode 0x04 (A key)
  • {HID 4} - Send keycode 4 (A key, decimal)
  • {HID 0x02 0x04} - Send Ctrl+A (0x02=Ctrl modifier, 0x04=A key)
  • {HID 2 4} - Send Ctrl+A (decimal modifiers and keycodes)
  • {HID 0x01 0x2A} - Send Shift+Backspace (0x01=Shift, 0x2A=Backspace)
  • {HID 0x08 0x17} - Send GUI+T (0x08=GUI/Windows key, 0x17=T)
  • {HID 0x02 0x16 0x17 0x18} - Send Ctrl+STU (multiple keys with modifier)
  • {HID 0xE2} - Send volume down (0xE2 consumer control)
  • {HID 0xE9} - Send volume up (0xE9 consumer control)
  • {HID 0xEA} - Send volume mute (0xEA consumer control)
  • {HID 0xB5} - Send next track (0xB5 consumer control)
  • {HID 0xB6} - Send previous track (0xB6 consumer control)
  • {HID 0xCD} - Send play/pause (0xCD consumer control)
  • HID accepts hex (0x prefix) or decimal values. Modifiers: 1=Shift, 2=Ctrl, 4=Alt, 8=GUI

Mouse Control Tokens

  • {SETMOUSE 400 300} - Set mouse to absolute position (400, 300)
  • {MOVEMOUSE 10 -5} - Move mouse 10 pixels right, 5 pixels up
  • {MOUSECLICK} - Left mouse click
  • {MOUSECLICK 2} - Right mouse click (1=left, 2=right, 3=middle)
  • {MOUSEDOUBLECLICK} - Double left click
  • {MOUSEPRESS 1} - Hold down left mouse button
  • {MOUSERELEASE 1} - Release left mouse button
  • {SETMOUSE {MATH 200 + 100 * cos({MATH {i} * 3.14159 / 180})} {MATH 200 + 100 * sin({MATH {i} * 3.14159 / 180})}} - Circular positioning

Math Tokens

  • {MATH 10 + 5} - Basic arithmetic (returns 15)
  • {MATH 3.14 * 2} - Floating point math (returns 6.28)
  • {MATH 100 / 3} - Division with decimals (returns 33.33)
  • {MATH cos(1.57)} - Cosine function (returns ~0)
  • {MATH sin(1.57)} - Sine function (returns ~1)
  • {MATH {i} * 2} - Using loop variables in calculations
  • {MATH cos({MATH {i} * 3.14159 / 180})} - Nested math expressions

Data Generation Tokens

  • {ASCII 65} - Send raw character 'A' (decimal 65)
  • {ASCII 0x41} - Send raw character 'A' (hex 0x41)
  • {ASCII {RAND 65 90}} - Send random uppercase letter
  • {RAND 1 100} - Generate random number between 1 and 100
  • {RAND {i} {i}} - Generate number equal to loop variable i
  • ASCII accepts decimal values or hex with 0x prefix

Variable Tokens

  • {i} - Loop variable 'i' value
  • {j} - Loop variable 'j' value
  • {counter} - Loop variable 'counter' value
  • Any variable name defined in LOOP statements

Escape Sequences

  • \n - Newline character
  • \t - Tab character
  • \r - Carriage return
  • \b - Backspace character
  • \f - Form feed character
  • \v - Vertical tab character
  • \\ - Literal backslash
  • \{ - Literal opening brace
  • \} - Literal closing brace

Simple Examples

  • Hello World{ENTER} - Type text and press Enter
  • {MOUSECLICK}{SLEEP 100}Test - Click mouse, wait, then type
  • {SETMOUSE 100 200}{MOUSECLICK} - Click at specific position
  • {MATH 5 + 3} = 8 - Simple calculation
  • Random: {RAND 1 10} - Random number 1-10
  • {LOOP i 1 1 3}{i} {ENDLOOP} - Outputs "1 2 3 "
  • {HID 0xE9} - Volume up
  • {ASCII 65} - Output 'A' character
  • {WIFI MyNetwork MyPassword} - Connect to WiFi

Complex Examples

  • {SETMOUSE 400 300}{LOOP i 0 1 360}{SETMOUSE {MATH 400 + 100 * cos({MATH {i} * 3.14159 / 180})} {MATH 300 + 100 * sin({MATH {i} * 3.14159 / 180})}}{SLEEP 10}{ENDLOOP} - Draw 100px radius circle
  • {MOUSEPRESS 1}{LOOP i 0 10 100}{SETMOUSE {MATH 100 + {i}} {MATH 100 + {i}}}{SLEEP 50}{ENDLOOP}{MOUSERELEASE 1} - Drag diagonal line
  • {LOOP i 32 1 126}ASCII {i}: {ASCII {i}}\n{ENDLOOP} - Print ASCII table
  • Password: {LOOP i 1 1 8}{ASCII {RAND 48 57}}{ENDLOOP} - 8-digit random password
  • {LOOP}{CHORD ALT+TAB}{SLEEP {RAND 2000 5000}}{ENDLOOP} - Random window switching
  • {LOOP i 1 1 10}Step {i}{ENTER}{SLEEP 1000}{BREAK i 5}{ENDLOOP} - Count to 5
  • {SETMOUSE 200 200}{MOUSEDOUBLECLICK}{SLEEP 500}{CHORD CTRL+A}Select All - Double-click and select all
  • {LOOP angle 0 5 360}{SETMOUSE {MATH 300 + 150 * cos({MATH {angle} * 3.14159 / 180})} {MATH 200 + 150 * sin({MATH {angle} * 3.14159 / 180})}}{MOUSECLICK}{SLEEP 20}{ENDLOOP} - Click points in circle
  • {HID 0x02 0x2C}{SLEEP 100}{HID 0x02 0x19} - Ctrl+Space, wait, Ctrl+V (paste special)
  • {LOOP i 1 1 5}{HID 0xE2}{SLEEP 500}{ENDLOOP} - Volume down 5 times with delay
  • {HID 0x01 0x15}{SLEEP 3000}{HID 0x01 0x08}{SLEEP 3000}{HID 0x01 0x16}{SLEEP 3000}{HID 0x01 0x16}{SLEEP 3000}{HID 0x01 0x1C}{SLEEP 3000}{HID 0x01 0x05} - Linux Magic SysRq REISUB sequence
  • {WIFI GuestNetwork password123}{SLEEP 5000}Connected to guest network! - Connect to WiFi and confirm

Keymap Editor

Custom keymaps remap characters to different HID key + modifier combinations, enabling correct typing on non-English keyboard layouts. The active keymap applies to all plain text output. The built-in en layout cannot be modified or deleted.

Char HID Key (dec) Modifier Mod flags Del
Modifier: 0=none 2=Shift 4=Alt 8=GUI 64=AltGr. Combine by adding.
Parses JSON and refreshes the table above.

Keymap JSON Format

Each entry maps a Unicode character to a HID key code and modifier byte. key is the raw HID usage ID (keyboard/keypad page 0x07) in decimal. mod is the modifier bitmask: 2=Shift, 4=Alt, 8=GUI, 64=AltGr. Values combine additively. 

{
  "id": "my-layout",
  "name": "My Custom Layout",
  "map": [
    {"char": "y", "key": 29, "mod": 0},
    {"char": "z", "key": 28, "mod": 0},
    {"char": "ä", "key": 52, "mod": 0},
    {"char": "Ä", "key": 52, "mod": 2},
    {"char": "@", "key": 20, "mod": 64}
  ]
}
